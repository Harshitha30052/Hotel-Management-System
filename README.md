# Hotel Management System

We developed this Hotel Management System as a group project to simplify hotel operations and provide a centralized platform for managing bookings and customer services digitally. The project focuses on improving user experience through room booking, authentication, food ordering, and booking management functionalities.

## Project Overview

The main objective of this project was to understand how a full-stack web application works by integrating frontend, backend, and database technologies into a single system.

This application allows users to register, log in, browse hotel services, book rooms, place food orders, and manage reservations. Administrative functionality was also implemented to manage booking information and customer records efficiently.

* **php/** – Contains backend logic and server-side processing
* **sql/** – Contains database scripts and schema files
* **images/** – Stores project images and assets
* **users.txt** – Stores user-related records
